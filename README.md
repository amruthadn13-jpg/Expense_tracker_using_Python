# Expense_tracker_using_Python
**💰 Expense Tracker in Python**

A simple Python program that allows users to enter daily expenses and calculates important financial insights such as total expense, highest expense, and lowest expense.

This project helps beginners practice Python loops, lists, and basic logic building.

**📌 Features**

Accepts multiple expense values from the user

Calculates the total expense

Finds the highest expense

Finds the lowest expense

Uses basic Python logic without built-in functions like max() or min()

**🧠 Concepts Used**

This project demonstrates the use of:

input() for user input

split() to separate values

Python lists

for loops

Type conversion using int()

Logical comparison (>, <)

Basic problem-solving approach

**⚙️ How It Works**

The user enters expenses separated by spaces.

The program splits the input into individual values.

Each value is converted from string → integer.

The program loops through the list to:

Calculate the total expense

Find the maximum expense

Find the minimum expense

The results are displayed to the user.

**▶️ Example**
Input
Enter the Expenses separated by space:
100 5000 600 8900 350
Output
Total Expense: 14950
Highest Expense: 8900
Lowest Expense: 100

**🚀 How to Run**

Install Python on your system.

Save the file as:

expense_calculation.py

Run the program:

python expense_calculation.py
